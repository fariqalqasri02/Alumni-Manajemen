# Use Case Sistem Manajemen Alumni

## Aktor

- User
- Admin

## Use Case User

- Registrasi akun baru
- Login ke sistem
- Melihat dashboard
- Mengelola profil pribadi
- Melihat lowongan kerja
- Mencari dan memfilter lowongan
- Melihat kegiatan karier
- Mendaftar kegiatan karier
- Mengisi tracer study
- Melihat notifikasi sistem

## Use Case Admin

- Login ke sistem
- Melihat dashboard admin
- Mengelola data lowongan kerja
- Mengelola data kegiatan karier
- Mengelola hasil tracer study
- Mengelola role dan hak akses pengguna
- Melihat laporan sistem
- Export laporan PDF
- Export laporan Excel
- Melihat statistik tracer study

## Alur Inti

### 1. Registrasi dan Login

- User mengisi nama, email, NIM, status pengguna, nomor telepon, dan kata sandi.
- Sistem membuat akun dan profil awal.
- User login dan diarahkan ke dashboard.

### 2. Lowongan Kerja

- Admin menambahkan lowongan.
- Sistem mempublikasikan lowongan dan membuat notifikasi.
- User dapat mencari, memfilter, lalu membuka detail lowongan.

### 3. Kegiatan Karier

- Admin menambahkan kegiatan.
- Sistem mempublikasikan kegiatan dan membuat notifikasi.
- User mendaftar kegiatan dan sistem menyimpan status pendaftaran.

### 4. Tracer Study

- Alumni mengisi status pekerjaan, masa tunggu kerja, gaji, dan relevansi kompetensi.
- Admin memantau data tracer study dan statistik hasilnya.

### 5. Laporan

- Admin membuka halaman laporan.
- Sistem menampilkan ringkasan data pengguna, lowongan, kegiatan, pendaftaran, dan tracer study.
- Admin mengekspor laporan ke PDF atau Excel.
